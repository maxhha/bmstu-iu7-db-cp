title: Создание аукциона

order: Пользователь, Сервер, Менеджер

_: **1. Создание аукциона**

Пользователь -> Сервер: `createAuction`

note:
**createAuction** создает новый аукцион в базе данных.

Пользователь <- Сервер: Идентификатор аукциона

_: **2. Заполнение информации**

Пользователь -> Сервер: `updateAuction`

note:
Изменение информации возможно только до начала проведения торгов.

Сервер -> Пользователь: Измененный аукцион
