title: Пополнение счета

order: Пользователь, Сервер, Банк, Менеджер

_: **1. Создание счета**

note:
При регистрации пользователю создаются счета на действующих номинальных аккаунтах, но при добавлении новых номинальных аккаунтов пользователь должен иметь возможность создать на них свои счета.

Пользователь -> Сервер: `createAccount`
Сервер -> Банк: `createAccount`
Банк -> Сервер: Номер счета
Сервер -> Пользователь: Созданный счет

_:**2. Пополнение счета**
Пользователь --> Банк: Создает денежный перевод по реквизитам номинального счета с указанием собственного счета

note Банк:
Добавляет транзакцию в базу данных

Банк -> Сервер: `newDepositTransaction`

Сервер --> Пользователь: Отправляет уведомление на телефон

end
