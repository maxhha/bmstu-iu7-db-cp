title: Создание товара

order: Пользователь, Сервер, Менеджер

_: **1. Создание товара**

Пользователь -> Сервер: `createProduct`

note:
**createProduct** создает новый товар в базе данных.

Пользователь <- Сервер: Идентификатор товара

_: **2. Заполнение информации**

Пользователь -> Сервер: `updateProduct`

Сервер -> Пользователь: Измененный товар

_: **3. Модерация товара**

Пользователь -> Сервер: `requestModerateProduct`

Пользователь <-- Сервер: Отправка токена на телефон
Пользователь -> Сервер: `approveModerateProduct(token)`
Пользователь <- Сервер: Измененный товар

...: Менеджер приступил к проверке товаров

Менеджер <- Сервер: `products`

if: Информация о товаре полна и корректна

Менеджер -> Сервер: `approveProduct`

Сервер --> Пользователь: Отправка уведомления на почту
Сервер --> Менеджер: Измененный товар

else: Предоставленная информация о товаре некорректна или неполная

Менеджер -> Сервер: `declineProduct`
Сервер --> Пользователь: Отправить уведомление на почту
Сервер --> Менеджер: Измененный товар

...: Клиент должен изменить информацию о товаре и отправить ее на повторную модерацию

end
